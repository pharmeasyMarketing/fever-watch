# Local-script city/state names - content-team review notes (2026-06-12)

`config/city_names_local.json` now carries native-script names for 223 of the 228 cities
(30 states/UTs), feeding the share-card sub-line ("{city}, {state}" in the state language)
in `src/build_share_cards.py`. The 5 NE cities (Shillong, Aizawl, Kohima, Itanagar, Imphal)
are intentionally absent and use the English card.

Provenance: generated by AI (one generator per language) and dual-verified by two
independent AI reviewers per language (2026-06-12). 222 of 223 city names passed both
reviewers; state names all passed. The language assignment per state is the content-team
approved v1 mapping (Goa, J&K, Ladakh, Sikkim, Assam, A&N, Chandigarh -> Hindi).
